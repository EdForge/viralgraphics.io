Class: Color

VG.Core.Color

new Color()

Creates a new VG.Core.Color which specifies a specific RGB color, including alpha. Internally the color values are represented normalized between 0 .. 1.
There are several constructors available:
  • The copy constructor takes a single VG.Core.Color value.
  • The HEX constructor takes a single string value of the form "#ffffff".
  • You can specify RGB values using either 3 arguments for red, green and blue or 4 values including alpha. All these values are between 0..255.
Internally all values are stored as normalized value between 0..1. If you want to pass normalized value to an constructor please use the VG.Core.NormalizedColor constructor.
