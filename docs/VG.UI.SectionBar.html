Class: SectionBar

VG.UI.SectionBar

new SectionBar()

Creates a VG.UI.SectionBar.
SectionBars can be added to the left or right side of an VG.UI.Workspace and can contain a list of vertically stacked VG.UI.SectionBarButtons. SectionBars cannot be resized. Use addItem() and addItems() to add items to the SectionBar.
