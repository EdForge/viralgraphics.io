Class: SectionToolBar

VG.UI.SectionToolBar

new SectionToolBar()

Creates a VG.UI.SectionToolBar.
SectionToolBars can be added to the left or right side of an VG.UI.Workspace and can contain a list of vertically stacked VG.UI.SectionToolBarButtons which are grouped into sections. SectionToolBars cannot be resized.
