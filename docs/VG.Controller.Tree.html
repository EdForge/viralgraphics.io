Class: Tree

VG.Controller.Tree

new Tree()

Creates an array controller which handles nested object hierarchies.
Note that index locations for this controller are not numbers but string, like "0.1" or "" to indicate the root level. You will not need to create this controller yourself, it is mostly returned by widgets like VG.UI.TreeWidget.
Supports the following observer keys: "selectionChanged", "changed", "addItem", "removeItem".
Properties:
Name Type Description
selected object The currently selected object.
selection array The currently selected objects when multiSelection is on.
multiSelection bool Set to true if this controller should handle multiple selections, default is false.
length number The number of items managed by the controller (read-only).
