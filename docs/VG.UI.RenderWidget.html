Class: RenderWidget

VG.UI.RenderWidget

new RenderWidget()

Creates an RenderWidget to perform arbitrary realtime 2D/3D rendering. The method render() is called to perform the rendering and needs to be overriden. The widget receives calls to render() in 60 fps per second.
