Class: Dialog

VG.UI.Dialog

new Dialog(title)

Creates an VG.UI.Dialog object. VG.UI.Dialog inherits from VG.UI.Window.
Dialogs are modal Windows, i.e. other user interface elements in the Workspace will be disabled while a Dialog is visible. Dialogs support a button row at the bottom of the Dialog. You can call addButton() and addButtonSpacer() to populate the Dialog with buttons. Dialogs can be shown by calling VG.UI.Workspace.showWindow.
Parameters:
Name Type Description
title string The title of the Dialog

Methods

addButton(text, func)

Adds a button to the Dialog. Buttons are added from the left to right. To close the window, the callback function of the button has to call this.close().
Parameters:
Name Type Description
text string The text of the button
func function The callback function which gets invoked when the button is pressed. Make sure to call this.close() in the callback if you want to close the Dialog

addButtonSpacer(space)

Adds horizontal space to the button layout. If space is defined, it adds the value of space in pixels, otherwise the space is set to expanding, meaning it will take as much space as available.
Parameters:
Name Type Description
space number The space to add to the button layout in pixels, if undefined, it will take as much space as available (expanding)
