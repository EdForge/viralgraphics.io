Class: StackedLayout

VG.UI.StackedLayout

new StackedLayout(child)

Arranges it's child objects stacked on top of each other. The current property defines the currently visible child object at the top of the stack. By default the first added child object is currently visible. Layouts can contain VG.UI.Widget based classes or other layouts.
Parameters:
Name Type Description
child object Optional, the object to add to the layout.
Properties:
Name Type Description
length number The number of child objects in this layout (read-only).
current object The currently visible child object.
