Class: Image

VG.Core. Image

new Image(value, height)

Creates a new VG.Core.Image class containing the color values of an image. Image Data is allocated via an Uint8Array typed array. The image data is allocated in the power of two of the width and height values to ensure direct WebGL / OpenGL support. realWidth and realHeight contain the power of the two sizes.
